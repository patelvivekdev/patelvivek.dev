## Syntax highlighting

In MDX you can do highlighting with [`sugar-high`](https://sugar-high.vercel.app/). It support Server Component by default.

````mdx
```jsx
function log({ title }) {
  if (title) {
    console.log('title', title);
  }
}
```
````

Simple function

```jsx
function log({ title }) {
```
